# Chemical News Crawler

The Chemical News Crawler is a project aimed at aggregating the latest news and developments in the field of chemistry. This tool allows users to stay updated on new research, innovations, and industry trends by crawling various chemical news websites and extracting relevant information.

## Features
- **Automated Crawling:** Automatically fetches news articles from multiple sources at predefined intervals.
- **Data Aggregation:** Compiles articles and categorizes them based on topics or keywords.
- **User-Friendly Interface:** Provides an easy interface for users to browse through the latest news articles.
- **Search Functionality:** Users can search for articles based on keywords or dates.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/GYHWH/chemical-news-bot.git
   ```
2. Navigate to the project directory:
   ```bash
   cd chemical-news-bot
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start the application:
   ```bash
   python app.py
   ```

## Usage
After starting the application, you can access it via your web browser. The home page will display the latest news articles.
